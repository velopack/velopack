# VelopackCSharpAvalonia

A sample application demonstrating the **Velopack** installation and auto-update framework
built with [Avalonia UI](https://avaloniaui.net/).

## Features

- Automatic update checking on startup
- Delta updates for *fast, bandwidth-efficient* patching
- Cross-platform support for Windows, macOS, and Linux

## System Requirements

| Component        | Minimum Version |
|------------------|-----------------|
| .NET Runtime     | 8.0             |
| Windows          | 10 or later     |
| macOS            | 12 Monterey     |
| Disk Space       | 50 MB           |

## Getting Started

1. Launch the application after installation
2. The app will automatically check for updates
3. When an update is available, click **Update Now**

> **Note:** The first launch may take a moment while the runtime initializes.
